# Duckling-Adventure
Interactive 3D adventure game built in Unity with custom character, physics-based collisions, and third-person camera controls.

## Technologies Used
- **Unity** — Game engine for 3D development  
- **C#** — Scripting for gameplay, controls, and game mechanics  
- **Blender** — Custom 3D character modeling and animation  
- **TinkerCAD** — Additional 3D modeling assets  

---

## Key Features
- Third-person player controls with physics-based collision detection  
- Smooth, dynamic camera system following the player  
- Custom 3D character created in Blender with animations integrated into Unity  
- Interactive game environment with collectable items and simple challenges  

---

## Project Structure
- `Assets/` — Scripts, scenes, prefabs, and textures  
- `ProjectSettings/` — Unity project configuration  
- `Packages/` — Unity packages and dependencies  
- `.gitignore` — Ensures temporary Unity files are not tracked
